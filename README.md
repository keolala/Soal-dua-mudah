 SoalDuaMudah
 Deskripsi Program

Program SoalDuaMudah adalah program sederhana berbasis Java yang berfungsi untuk menentukan apakah sebuah angka merupakan bilangan genap atau ganjil.

Program ini menggunakan operator modulus (%) untuk memeriksa sisa hasil bagi dari suatu bilangan ketika dibagi 2.

Jika sisa bagi = 0, maka angka tersebut genap.

Jika sisa bagi ≠ 0, maka angka tersebut ganjil.

⚙️Cara Kerja Program

Program meminta pengguna untuk memasukkan sebuah angka.

Program memeriksa apakah angka tersebut habis dibagi 2 atau tidak:

if (angka % 2 == 0) {
    System.out.println(angka + " adalah angka genap");
} else {
    System.out.println(angka + " adalah angka ganjil");
}
